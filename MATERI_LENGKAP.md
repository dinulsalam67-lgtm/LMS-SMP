## 3. Persamaan Linear ğŸ“

### Konten yang Perlu Ditambahkan:

#### A. Pengertian PLSV
- **Definisi**: Persamaan dengan satu variabel berpangkat satu
- **Bentuk umum**: ax + b = c
- **Contoh**: 2x + 5 = 13, 3y - 7 = 8

#### B. Cara Menyelesaikan PLSV (dengan Gambar)
```
[GAMBAR: Timbangan/Neraca]
Konsep Keseimbangan:
- Apa yang dilakukan di ruas kiri, harus dilakukan di ruas kanan
- Tujuan: Isolasi variabel
```

**Langkah-langkah**:
1. Kumpulkan suku variabel di satu ruas
2. Kumpulkan konstanta di ruas lain
3. Sederhanakan
4. Bagi dengan koefisien variabel

**Contoh Visual**:
```
2x + 5 = 13
2x + 5 - 5 = 13 - 5  (kurangi 5 di kedua ruas)
2x = 8
2x/2 = 8/2  (bagi 2 di kedua ruas)
x = 4
```

#### C. Aplikasi PLSV
1. **Masalah Umur**
2. **Masalah Uang**
3. **Masalah Jarak, Kecepatan, Waktu**
4. **Masalah Campuran**


#### D. Ukuran Pemusatan Data

**1. Mean (Rata-rata)**
```
Rumus: xÌ„ = (Î£x) / n
Contoh: 5, 7, 8, 6, 9
Mean = (5+7+8+6+9)/5 = 35/5 = 7
```

**2. Median (Nilai Tengah)**
```
Langkah:
1. Urutkan data
2. Jika n ganjil: median = data tengah
3. Jika n genap: median = rata-rata 2 data tengah

Contoh: 3, 5, 7, 9, 11
Median = 7
```

**3. Modus (Nilai Paling Sering)**
```
Contoh: 4, 5, 5, 6, 7, 5, 8
Modus = 5 (muncul 3 kali)
```

#### E. Ukuran Penyebaran Data

2. **Kuartil**: Q1, Q2 (median), Q3
3. **Simpangan**: Rata-rata simpangan, simpangan baku
